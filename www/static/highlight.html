Oak highlight proxy

The highlight proxy syntax highlights Oak source code found elsewhere on the Web, for example on GitHub, using the same interface used to show the standard library source on this website.

The highlight proxy can also highlight just a portion of a larger source file, by specifing a range of lines using either start=N&end=M query parameters, or using line ranges specified in GitHub file URLs.

For an example, try pasting in the URL to this "Hello World" Oak program.
